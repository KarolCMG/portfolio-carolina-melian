## 🚀 **Inicio Rápido**

### **Desarrollo Local:**
```bash
npm install
npm run dev
# Abrir: http://localhost:5173
```

### **Despliegue en Internet:**
```bash
npm run build
# Subir carpeta 'dist' a Netlify
# URL: https://carolina-melian-portfolio.netlify.app
```
